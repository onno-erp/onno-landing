# onno landing

Landing page for [onno](https://onno.su).

Project-wide visual and interaction conventions are documented in [STYLEGUIDE.md](./STYLEGUIDE.md).

## Development

```sh
npm install
npm run dev
```

## Production image

Pushes to `main` publish `ghcr.io/onno-erp/onno-landing`. Flux deploys timestamped image tags to `onno.su`.
